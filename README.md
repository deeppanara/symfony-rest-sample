# symfony5-sample
Sample codes demonstrating Symfony, Doctrine, PHPUnit, etc. 

## Docs

* [Building Restful APIs with Symfony 5 and PHP 8](./docs/rest.md)

## Build 

Start a Postgres database.

```
docker compose up postgres
```

Run the application

```
symfony serve
```
